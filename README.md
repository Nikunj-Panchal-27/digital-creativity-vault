# ⚙ Installation

Clone the repository:

```bash
git clone https://github.com/Nikunj-Panchal-27/digital-creativity-vault.git
```

Go to the project directory:

```bash
cd digital-creativity-vault
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser:

```
http://localhost:3000
```
